Module 4: Biomedical AI Pipeline Interactive Demonstration
Overview
Welcome to the interactive companion for Module 4 Week 1 Notebook 1 – Shared biomedical artificial intelligence vocabulary.

This demonstration brings the concepts from your Jupyter Notebook to life. You do not need to write or execute any code to use this tool. Instead, you will use interactive sliders and dropdowns to manipulate data, train a machine learning model, and visualize how artificial intelligence makes decisions in real-time.

How to Use This Demonstration
This application is designed to be an interactive sandbox. As you progress through your notebook, use this app to visualize the mathematical concepts happening behind the scenes.

1. Choose Your Perspective
In the sidebar, you can toggle the demonstration's perspective between Clinical Care and Foundational Science. Notice how the exact same machine learning pipeline and dataset can be interpreted differently depending on whether your goal is treating patients at the bedside or discovering novel biological mechanisms in a lab.

2. Navigate the Pipeline
Use the sidebar navigation to move through the three core stages of the machine learning pipeline:

Activity 1: Data Preprocessing and Splitting: Explore the raw eICU dataset. Use the interactive slider to adjust the Train/Test split ratio and instantly see how the data is partitioned.

Activity 2: Model Training and Interactive Prediction: Train a Decision Tree algorithm. You can adjust the "Max Depth" of the tree to see how it changes the complexity of the model's rules. Then, use the Live Interactive Simulator to input custom patient metrics and watch the model predict the outcome in real-time based on the visual pathways it learned.

Activity 3: Cross-Validation: A single Train/Test split can be unreliable. Adjust the number of cross-validation folds to see how evaluating the model across multiple different data splits provides a more stable and accurate measure of its true performance (Accuracy, Sensitivity, Specificity, and Precision).

Alignment with Your Coursework
At the bottom of each activity in this demonstration, you will find expandable "Expected Insights" sections. These are designed to guide your thinking and help you answer the reflection questions presented in your Jupyter Notebook.

Use the visualizations generated here to support the answers you submit in Canvas.
